2D Interactable Environment

This project is a PC/laptop application featuring a 2D interactive Christmas-themed environment designed to spread positivity. It allows users to personalize and edit messages, making it a fun and creative way to share holiday cheer. Being part of this project has been a fulfilling experience for me, as it not only brings joy to users but also showcases the power of interactive design in fostering meaningful connections. I'm truly happy and proud to have contributed to something that spreads warmth and positivity during the holiday season.
